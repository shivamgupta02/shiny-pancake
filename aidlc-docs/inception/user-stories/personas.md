# User Personas — Expense Calculator App

## Persona 1: Rahul (Primary User)

| Attribute | Details |
|-----------|---------|
| **Name** | Rahul |
| **Age** | 28 |
| **Role** | Working professional (personal finance user) |
| **Tech-Savviness** | Average — uses apps daily, prefers simple guided interfaces |
| **Goals** | Track daily expenses, understand spending patterns, stay within budget |
| **Frustrations** | Forgets to log expenses, loses track of small purchases, surprised by monthly totals |
| **Device** | Mid-range Android phone |
| **Behavior** | Receives bank SMS alerts for transactions, prefers quick actions over complex setups |

### Key Characteristics
- Uses UPI and card payments regularly, receives transaction SMS from multiple banks
- Wants to see where money goes each month without manual effort
- Values simplicity — if it takes more than 2 taps, he'll skip it
- Reviews finances weekly/monthly but not daily
- Shares household expenses with family (future consideration)

## Persona 2: Priya (Secondary User — Future Multi-User)

| Attribute | Details |
|-----------|---------|
| **Name** | Priya |
| **Age** | 32 |
| **Role** | Family budget manager |
| **Tech-Savviness** | Average |
| **Goals** | Track household expenses, generate reports for family budgeting |
| **Frustrations** | No visibility into total family spending, manual spreadsheets are tedious |
| **Device** | Android phone |
| **Behavior** | Exports data monthly to share with spouse, uses categories to plan budgets |

### Key Characteristics
- Manages household finances for the family
- Needs clear reports and export functionality
- Values category organization and monthly summaries
- Represents future multi-user/family sharing use case
